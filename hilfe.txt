*** O P T I O N S S C H A L T E R
🎯 PFICHTANGABEN
Option	Kurz	Beschreibung
--mask	-m	PFAD ZUR PNG-VORLAGE - Das Bild, das als Maske dient (Schwarz/Weiß ideal)
--output	-o	NAME DER AUSGABEDATEI - z.B. "ergebnis.png" oder "mosaik.jpg"
📁 ORDNER-OPTIONEN
Option	Kurz	Beschreibung	Standard
--input-folder	-i	Input-Ordner mit den Quellbildern	./input
--output-folder	-of	Output-Ordner für das Ergebnis	Skript-Verzeichnis
🚀 SPEZIAL-MODI
Option	Kurz	Beschreibung	Effekt
--preview	-p	VORSCHAU-MODUS - Schnelle Berechnung	Tile-Größe: 40×25, Tiles: 500, kein Postprocessing
--high-quality	-hq	HOCHQUALITATIV-MODUS - Maximale Qualität	Tile-Größe: 250×160, Tiles: 15000, max. Schärfe & Farbe
📐 GRID-GRÖSSE
Option	Kurz	Beschreibung	Standard
--tiles	-t	Maximale Anzahl Tiles	3000
--width	-w	Manuelle Grid-Breite (optional)	Automatisch
--height	-hi	Manuelle Grid-Höhe (optional)	Automatisch
🧩 TILE-GRÖSSE
Option	Beschreibung	Standard
--tile-width	Breite jedes einzelnen Tiles in Pixeln	100
--tile-height	Höhe jedes einzelnen Tiles in Pixeln	64
⚔️ SCHÄRFE-OPTIONEN
Option	Kurz	Beschreibung	Standard
--contrast	-c	Kontrastverstärkung (1.0-3.0)	2.5
--edge-thickness	-et	Dicke der Kantenlinie in Tiles	2
--edge-multiplier	-em	Kanten-Kontrast Multiplier	3.0
--threshold	-th	Helligkeitsschwellwert (0-255)	128
--sharpen	-s	Nachschärfungs-Faktor	2.0
🎨 FARB-OPTIONEN
Option	Kurz	Beschreibung	Standard
--saturation	-sat	Sättigungsfaktor (0.0-2.0)	1.3
--vibrance	-vib	Vibrance-Faktor (intelligente Sättigung)	1.2
--black-level	-bl	Schwarzpunkt (0-255, niedriger = mehr Tiefe)	0
--white-level	-wl	Weißpunkt (0-255, niedriger = mehr Kontrast)	245
⚡ SCHALTER (BOOLEAN-OPTIONEN)
Option	Effekt wenn gesetzt
--no-contrast	Kontrastverstärkung AUSSCHALTEN
--no-edge	Kantenschärfung AUSSCHALTEN
--no-binary	Binäre Trennung AUSSCHALTEN (weichere Übergänge)
--no-morphology	Morphologische Reinigung AUSSCHALTEN
--no-postprocess	Nachbearbeitung AUSSCHALTEN
--no-color	Farbverstärkung komplett AUSSCHALTEN
--no-levels	Levels-Anpassung AUSSCHALTEN
--no-curves	Curves-Anpassung AUSSCHALTEN
🐛 DEBUG
Option	Kurz	Beschreibung
--debug	-d	Debug-Modus: Speichert zusätzliche Informationen (z.B. Masken-Visualisierung)
--version	-v	Zeigt Versionsnummer
